# Criptografía - Ejercicios en Python

Este repositorio contiene una colección de ejercicios prácticos de criptografía desarrollados en Python como parte de la asignatura de Criptografía.

## Estructura del repositorio

```
AE1/
 ├── Ejercicio01.py
 ├── Ejercicio02.py
 ├── ...
 ├── Ejercicio12.py
```

Los ejercicios están organizados por actividades evaluables (AE). Cada archivo corresponde a un ejercicio independiente.
